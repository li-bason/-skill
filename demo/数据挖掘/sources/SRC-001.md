    同一条记录不合理地多次出现。可能由于现实操作过程中错误地发生了多次数据记录操作，也
可能是一个样本同时归属多个针对不同样本类别地数据库，导致数据合并后出现重复
1. 数据重复
    数据值不一致。同一个事件在不同系统中出现不同的记录值。这其中包含不同系统的记录错误
问题，不同系统的更新机制、更新及时性问题，还有不同系统在管理流程中的作用和先后顺序差异
2. 数据不一致
    数据标准不一致。同一项数据不同系统可能采用不同的记录规则。常见的情况包括：不同国家
的系统用不同时区、数据记录的编码标准不一致、数据单位不一致。

偏差—方差权衡
• 低偏差和低方差是衡量模型性能的两个基本要素，但

学习中广为人知的偏差-方差权衡。
• 偏差（bias）反映了模型对目标变量的预测偏离真实
值的程度；而方差（variance）则表示模型对数据中
的噪声敏感的程度，通常与模型的复杂度有关。
• 在偏差-方差权衡中，我们力求找到一个模型，使其既
能有效学习数据的模式，又能避免过度拟合，从而在
新数据上有较好的表现。

• boosting方法的原理与bagging方法类似：我们建立一系列模型，然后将这些模型聚合起来，得到一个

K最近邻（k-Nearest Neighbor Classification，KNN）算法是机器学

基本思想：想知道新样本属于什么类别，就看和它最相似的样本是什么

## Page 150

q 选定一个K值，用于确定参考几个相似样本（K最近邻名字来源）
q 确定一个距离计算方法（最常用是欧氏距离）用于计算两个样本间的距离
q 对于每个新样本（测试数据），计算该样本与所有训练样本(历史样本)的距离。
q 根据距离对所有训练样本进行排序，选取距离最小（即最相似）的K个样本
q 统计K个样本中不同类别占比，选占比最大的类别作为该测试样本类别

以类聚”，可以找到离圆点最近的几个点，以它们中的大多数点的
类别决定新数据所属的类别。如果k = 3，由于圆点近邻的3个样本
中，三角形占比2/3，则认为新数据属于三角形类别。同理，k = 5，
则新数据属于正方形类别。
