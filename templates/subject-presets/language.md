# language preset

- 默认路线：只做语言积累与增量整理，不生成练习题，不设置每考点题量。
- 默认文件：`词汇语法积累表.md`、`翻译积累表.md`、`写作积累表.md`，三个表格独立维护。
- 不创建阅读板块、阅读策略或思维导图。
- 单词/词组查询：词性使用英文缩写，含义使用英英释义；使用 `## 词条：...` Markdown 分组，
  每个词条标题只出现一次，每个义项使用独立表格行并对应一个搭配和例句。
- 英译中：最长词组优先拆分，标注词汇表内/外，解释关键项后翻译，再询问是否收录新词条。
- 中译英：先拆中文语义单元，再映射英文单词/词组并组合译文，随后执行相同的确认收录。
- 翻译表使用 `## 原句：...` Markdown 分组，原句标题、拆分和译文只出现一次；关键表达与说明
  使用二列表格，一项一行。
  每个表外关键表达都必须询问是否加入词汇表。
- 翻译原句只有在用户要求保存时才进入翻译积累表。
- 写作部分只积累英文句子；先解释中文含义、句型结构和可替换部分，再询问是否收录。
- 长期更新：词汇按规范化词条、翻译按规范化原句、写作按规范化英文句子去重；已有主项只
  增补子项或缺失字段，不重复建组。
