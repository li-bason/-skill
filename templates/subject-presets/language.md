# language preset

- 默认路线：按板块整理，不走刷题主路线
- 默认文件：复习总结.md
- 默认板块：词汇语法 → 阅读 → 翻译 → 写作
- 板块缺失：由 skill 自动重组
- 写法：分类整理重写
- 统一结构：板块 → 原子规则或策略 → 考查要求 → 规则/例句/错误/步骤/表达素材/检查清单分块
- 范文、例句、模板句：压成表达素材点
- 练习题：主要转成板块总结
- 用户要求练习题时，选择题选项逐行并保留 Markdown 空行，答案按稳定 ID 独立分块
- 板块资料不足：按统一缺口阈值自动分流；大量缺失时直接用模型知识补足语法规则、策略、方法和写作框架并标记 `inferred`，不逐站联网
- 长期更新：只更新受影响板块；板块内去重重整
