# 质量检查

## 硬性错误

- skill 已触发但没有执行完整阶段链，或合同未设置 `full_pipeline_required: true`。
- 用户指出一个样例问题后只修该样例，没有扫描全部产物中的同类问题。
- `review_spec.md`、`review_lock.yaml`、`progress-log.md`、原子点数量、题量或正式产物互相不同步。
- 合同、manifest、progress log 或合同声明的产物缺失。
- 无法解析合同，模板 ID 不存在或版本不匹配。
- 最细知识点/遗漏点无法识别，因而不能验证最低题量。
- 每点少于 3 题，题目 ID 重复，或题目与答案 ID/顺序不一致。
- 期末文理工缺少思维导图；语言类缺少四个板块；竞赛缺少已考点/遗漏点。
- 声称来自教材/原题的内容没有对应来源 ID。
- 用户给定课程名/大纲与正式范围不一致，或将标记为 `excluded` 的材料用于确定题型、章节或必考点。
- 有官方题型蓝图但题型数量、总分或考点覆盖表不匹配。
- 用户直接给出的实质大纲、题型或范围仅存在于 `review_spec.md`/合同中，而没有对应的 `kind: user_text` 来源记录。
- 缺少 `atomic_points.md`，或并列的定义/流程/计算要求没有拆成独立原子考点。
- 知识点以摘要替代来源原句；必背内容缺少来源定位，或与 `sources/` 对应文字不一致。
- 将指定页整页复制进单个知识点，或混入不能直接回答该考点的背景、案例、目录和重复内容。
- 不同知识点版式不统一，或缺少“原子点标题、考查要求、分块内容”中的任一部分。
- 同一必背内容中的目标、定义、区别、步骤、优缺点或参数连续堆叠，没有分段和空行。
- 资料只列术语名称、章节目录或题型，而知识点没有补充定义、公式、判定条件或步骤。
- 算法流程、实现步骤或计算过程被塞进表格，没有使用纵向有序步骤和独立公式块。
- 正式题库仍包含“完整说明……定义、核心内容和作用”“围绕……逐项完成”“结合一个场景说明
  如何运用”等骨架题干。
- 选择题缺少有效选项或干扰项；计算题没有数值条件；同一原子点三题只是重复提问；答案只有
  “评分要点”或重复粘贴来源，没有给出具体结论。
- 选择题以“是否符合课程材料”为题干，或把章节标题、题型目录、其他考点摘录作为干扰项；
  四个选项不在同一语义层级。
- 选择题的题干与选项没有分行，或 `A：`、`B：`、`C：`、`D：` 未各占一行。
- Markdown 预览中选择题选项仍处于同一段；题干与 A、A/B、B/C、C/D 之间缺少空行。
- 答案使用重复的 `1、2、3` 编号，或没有按 `question_id｜题型` 分块显示。
- 图中坐标、权重示意值、节点输入输出、箭头标签、页码或水印混入必背内容。
- 正式产物包含 `??`、问号变量、零宽字符、乱码数学字形或从 PDF 错位抽取的上下标。
- 已发现材料缺口或无法解析内容，却没有视觉复核记录；视觉复核仍失败时，没有权威 Web
  补充记录或 `unverified` 标记。
- 使用了联网内容但 manifest 缺少 URL、标题、访问日期、`external` 状态或对应原子点。
- 训练题库任一原子考点少于 3 题。
- 只拆分或修复用户举例的部分，大纲其余条目仍使用复合标题。
- `atomic_points.md` 未覆盖全部上级大纲条目，或“几个概念”没有按来源中列出的实际概念继续拆分。
- 将两个或更多不同方法、算法、任务、指标合并成一个原子点，导致来源内容串在同一段。
- 题库总题数小于 `原子考点数 × minimum_questions_per_point`。

## 警告

- 输入材料存在 `needs_conversion`、`needs_ocr` 或 `failed`。
- 存在 `unverified`、低置信度或来源冲突。
- 难度比例偏离 40/40/20 超过 15 个百分点。
- PDF 因可选依赖或系统库缺失而未生成。

## 输出

运行 `scripts/validate_review.py <任务目录> --json` 和
`scripts/validate_output_structure.py <任务目录> --json`，并将结果保存为
`quality_report.json`。error 为零才允许交付；warning 必须在交付摘要中披露。
修改模板或生成器后还必须运行 `scripts/test_golden_structure.py`。

视觉验收需要截图时，临时目录按需创建；保存必要检查结果后删除临时截图、中间 PDF、OCR
缓存和空目录。`tmp/`、`tmp_pdf_check/` 不属于正式产物，不得进入压缩包或后续常规扫描。
