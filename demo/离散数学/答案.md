### AP-008 重言式、矛盾式与可满足式

#### Q-022｜选择题

- **答案：** D

- **解析：** 重言式（永真式）：公式在所有赋值下都为真。

- **来源：** `inferred（模型知识补全）`

#### Q-023｜填空题

- **答案：** 矛盾式（永假式）

- **完整表述：** 矛盾式（永假式）：公式在所有赋值下都为假。

- **来源：** `inferred（模型知识补全）`

#### Q-024｜简答题

**参考答案：**

- 重言式（永真式）：公式在所有赋值下都为真。
- 矛盾式（永假式）：公式在所有赋值下都为假。
- 可满足式：至少存在一个赋值使公式为真；重言式也属于可满足式。
- A 为重言式 ⇔ ¬A 为矛盾式
- A 为矛盾式 ⇔ ¬A 为重言式
- A 可满足 ⇔ ¬A 不是重言式

**来源：** `inferred（模型知识补全）`

### AP-009 常用等值式

#### Q-025｜选择题

- **答案：** A

- **解析：** ¬¬p ⇔ p

- **来源：** `inferred（模型知识补全）`

#### Q-026｜判断题

- **结论：** 正确

- **订正：** p → q ⇔ ¬p ∨ q

- **来源：** `inferred（模型知识补全）`

#### Q-027｜简答题

**参考答案：**

- ¬¬p ⇔ p
- p → q ⇔ ¬p ∨ q
- p ↔ q ⇔ (p → q) ∧ (q → p)
- ¬(p ∧ q) ⇔ ¬p ∨ ¬q
- ¬(p ∨ q) ⇔ ¬p ∧ ¬q
- p ∨ (q ∧ r) ⇔ (p ∨ q) ∧ (p ∨ r)
- p ∧ (q ∨ r) ⇔ (p ∧ q) ∨ (p ∧ r)
- p ∨ ¬p ⇔ 1；p ∧ ¬p ⇔ 0
- p ∨ 0 ⇔ p；p ∧ 1 ⇔ p
- p ∨ p ⇔ p；p ∧ p ⇔ p

**来源：** `inferred（模型知识补全）`

### AP-016 归结原则证明

#### Q-046｜选择题

- **答案：** D

- **解析：** (L∨A) ∧ (¬L∨B) ⇒ A∨B

- **来源：** `inferred（模型知识补全）`

#### Q-047｜填空题

- **答案：** 证明结论

- **完整表述：** 证明结论 C 时，把 ¬C 加入前提子句集；若最终归结出空子句 □，则 C 成立。

- **来源：** `inferred（模型知识补全）`

#### Q-048｜简答题

**参考答案：**

- (L∨A) ∧ (¬L∨B) ⇒ A∨B
- 证明结论 C 时，把 ¬C 加入前提子句集；若最终归结出空子句 □，则 C 成立。
- 前提：(p→q)→r，r→s，¬s；结论：p∧¬q。
- 否定结论：¬(p∧¬q) ⇔ ¬p∨q。
- 化为子句：p∨r，¬q∨r，¬r∨s，¬s，¬p∨q。
- 由 p∨r 与 ¬p∨q 归结得 q∨r。
- 由 q∨r 与 ¬q∨r 归结得 r。
- 由 r 与 ¬r∨s 归结得 s。
- 由 s 与 ¬s 归结得空子句 □，所以原结论成立。

**来源：** `inferred（模型知识补全）`
