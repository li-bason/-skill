# liberal-arts preset

- 默认骨架：框架提纲 → 题目 → 答案
- 知识点结构：章节 → 原子考点 → 考查要求 → 概念/背景/原因/内容/影响/评价/论据分块
- 每个最细知识点至少 3 题
- 默认补题：选择 + 简答混合
- 默认答案：`question_id｜题型` → 考点 → 答题提纲 → 参考答案 → 来源
- 选择题选项逐行并保留 Markdown 空行；答案禁止循环 1、2、3 编号
- 题型缺失：必须追问用户
- 长期更新：按最新资料整体重整
- 资料只给标题或目录时：直接用模型知识补齐可作答的概念、事实链、观点、论据和答题层次，标记 `inferred`
- 禁止：把章节目录、题型提示、其他考点段落或空泛关键词当作知识正文
