# 文理工三文件骨架 v1

## atomic_points.md

```markdown
| atomic_id | 上级条目 | 原子考点 | 考查动作 | 来源 | 训练题数 |
|---|---|---|---|---|---|
| AP-... | ... | ... | 定义/流程/计算 | SRC-001#Page ... | ≥3 |
```

## 知识点.md

```markdown
# <科目>期末复习知识点
## <上级大纲编号>
### <ATOMIC-ID>｜<序号>. <原子考点>

| 考点编号 | 考查要求 | 来源 |
| --- | --- | --- |
| `<ATOMIC-ID>` | <定义/比较/流程/计算> | `<SRC-ID · Page>` |

#### 必背内容

| 要点 | 内容 |
| --- | --- |
| **目标** | <直接回答目标的内容，并对关键术语加粗> |
| **分类区别** | <直接回答区别的内容，并对关键术语加粗> |
```

不得复制整页，不得混入背景故事、图表标签、同页其他考点或重复案例。
不同语义项必须各占一行，不能连续堆在同一个单元格中。

流程或计算型改用：

```markdown
#### 步骤与计算

公式：<独立公式块>

1. <步骤一>
2. <步骤二>
3. <步骤三>

结果：<结论或判断>
```

## 思维导图.md

使用 Mermaid `mindmap`，节点 ID/文字与知识点树一致。

## 题目.md

每个原子考点标题下至少 3 道题：

```markdown
### <ATOMIC-ID> <原子考点>
<!-- question_id: Q-... atomic_id: <ATOMIC-ID> difficulty: basic source: generated -->
1. 【选择】<题干>

A：<选项 A>

B：<选项 B>

C：<选项 C>

D：<选项 D>
```

## 答案.md

与题目 ID 和顺序一致，包含原子考点、思路、关键步骤和答案。教师蓝图仅用于调整题型
覆盖比例，不得挤占每原子考点训练题量或额外创建未要求的产物。

```markdown
### <ATOMIC-ID> <原子考点>

<!-- answer_id: Q-... atomic_id: <ATOMIC-ID> -->
#### Q-...｜选择题

- **答案：** A

- **解析：** <具体解析>

- **来源：** `<SRC-ID>`
```

禁止用循环的 `1、2、3` 作为答案标题。
