## Page 1
Data preprocessing is divided into four stages: data cleaning, data integration, data reduction, and data transformation.

High-quality data supports better models and predictions. Accuracy, completeness, and consistency are three main factors contributing to data quality.

Data cleaning removes outliers, replaces missing values, smooths noisy data, and corrects inconsistent data. Data integration combines data from multiple sources. Data reduction produces a smaller representation while maintaining the integrity of the original data. Data transformation converts data into a form appropriate for modeling.
